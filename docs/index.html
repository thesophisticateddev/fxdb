STEP 2 ‚Äî Build
# Requires Java 17+ and Maven 3.8+

// Usage

Four Steps to Query

01
Add a Connection

Go to File ‚Üí New Connection. Enter host, port, user, and password. FXDB encrypts credentials automatically.

02
Browse Schema

Select a connection from the left panel. Expand databases, schemas, and tables in the explorer tree.

03
Execute Queries

Type SQL in the editor panel and hit Run. Results appear instantly in a paginated result grid.

04
Manage & Repeat

Edit or remove connections with a right-click. Switch between multiple live connections without disconnecting.
